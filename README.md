#  Hospital Management System Using C++

##  Project Overview

The Hospital Management System is a console-based application developed in C++ to manage patient records efficiently. The system provides a simple and user-friendly menu-driven interface that allows users to perform various patient management operations such as adding, displaying, searching, updating, and deleting patient records.

This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts and the use of the Standard Template Library (STL) in C++.

---

##  Objectives

- To understand Object-Oriented Programming concepts.
- To implement CRUD (Create, Read, Update, Delete) operations.
- To develop a menu-driven application in C++.
- To improve programming and problem-solving skills.

---

- Add Patient Record

- Search Patient by ID

##  How to Run

1. Open the project in Visual Studio Code or Code::Blocks.
2. Compile the `main.cpp` file.
3. Run the program.
4. Select the desired option from the menu.
